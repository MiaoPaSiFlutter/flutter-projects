## 执行命令
 - 默认命令
    `sh hzy_rob_project.sh`
 - 获取帮助
    ```
    sh hzy_rob_project.sh -h 
    sh hzy_rob_project.sh —help
    ```
 - 配置参数
    ```
    sh hzy_rob_project.sh -ocom.example.example -nexample
    sh hzy_rob_project.sh --org=com.example.example --name=example
    ```
